## 1. 首页 `/`

**对应截图**：图2

### 页面结构

```
首页
├── 头部区域（渐变背景色）
│   ├── 标题：卿豪网络，你想要的，这里都有
│   ├── 副标题：担保交易·累计交易超过xxx单
│   └── 搜索框：输入游戏名称 + 搜索按钮
├── 热门搜索标签（如：向僵尸开炮）
└── 热门游戏网格
    ├── 标题：热门游戏
    └── 游戏卡片（4列网格，图标+名称）
```

### 功能点

- [ ] 头部渐变背景（蓝绿色渐变）
- [ ] 搜索框点击跳转 `/games` 并带搜索词
- [ ] 热门搜索标签点击直接搜索
- [ ] 游戏卡片点击跳转 `/goods-list/:gameId`
- [ ] 网格布局 4 列，图标圆角正方形

### 交互说明

- 点击游戏卡片 → 跳转到该游戏的商品列表页
- 点击搜索框 → 跳转到全部游戏页，唤起搜索
- 点击热门标签 → 跳转到全部游戏页并执行搜索

## 3. 商品列表 `/goods-list/:gameId`

**对应截图**：图3

### 页面结构

```
商品列表
├── 导航栏（返回 + 游戏标题 + 分享）
├── 搜索框（请输入关键词查找 + 搜索按钮）
├── 筛选栏
│   ├── 客户端
│   ├── 安卓
│   ├── 排序
│   └── 筛选
├── 商品列表
│   └── 商品卡片
│       ├── 左侧截图
│       │   └── 图片数量角标
│       ├── 右侧信息
│       │   ├── 平台 + 服务器 + 系统
│       │   ├── 标签（找回包赔/官方验号/安全保障）
│       │   ├── 价格（¥xxx.xx）
│       │   ├── 一键复制按钮
│       │   └── 客服咨询按钮
│       └── 标题
└── 底部提示（暂无更多）
```

### 功能点

- [ ] 顶部显示当前游戏图标和名称
- [ ] 筛选栏（客户端/平台/排序/筛选）
- [ ] 商品卡片展示（截图、标签、价格、操作按钮）
- [ ] 图片数量角标（如 13图、4图）
- [ ] 价格红色加粗显示
- [ ] "一键复制"按钮（复制商品信息）
- [ ] "网站唯一咨询客服"按钮

### 交互说明

- 点击商品卡片 → 跳转到商品详情页 `/goods-detail/:goodsId`
- 点击筛选栏 → 弹出筛选选项（当前 mock 阶段可做静态展示）
- 点击"一键复制" → 复制商品标题/ID 到剪贴板
- 点击"咨询客服" → 调起客服（后续接入）

## 4. 商品详情 `/goods-detail/:goodsId`

**对应截图**：图4

### 页面结构

```
商品详情
├── 导航栏（返回 + 游戏标题 + 分享）
├── 截图轮播区
│   └── 游戏截图滑动浏览
├── 价格信息
│   ├── 价格（¥xxx.xx 红色大字）
│   └── 标题
├── 元信息
│   ├── 上架时间
│   └── xx+人想要
├── 商品ID区域
│   ├── 商品ID + 一键复制按钮
│   └── 提示：点击复制，粘贴发送给最下方客服
├── 商品参数表格
│   ├── 参数列表（3列：标签-值-标签-值...）
│   └── "全部参数"展开按钮
├── 游戏截图详情
│   └── 大图展示（多张）
└── 底部固定栏
    └── 客服提示：卿豪网络（请勿咨询他人以免被骗）
```

### 功能点

- [ ] 截图横向滑动浏览（Swiper 效果）
- [ ] 价格大字红色显示
- [ ] 商品 ID 一键复制
- [ ] 商品参数表格（多列展示）
- [ ] "全部参数"点击展开更多
- [ ] 底部固定客服提示栏

### 交互说明

- 左右滑动 → 浏览截图
- 点击"一键复制" → 复制商品ID到剪贴板，Toast 提示"复制成功"
- 点击"全部参数" → 展开/收起完整参数
- 点击客服栏 → 调起客服（后续接入）

## 公共组件清单

| 组件名 | 用途 | 使用页面 |
|--------|------|----------|
| NavBar | 顶部导航栏（返回+标题+分享） | 全部游戏、商品列表、商品详情 |
| SearchBar | 搜索框 | 首页、全部游戏、商品列表 |
| GameCard | 游戏卡片（图标+名称） | 首页、全部游戏 |
| GoodsCard | 商品卡片 | 商品列表 |
| Tag | 标签（找回包赔/官方验号等） | 商品列表、商品详情 |
| EmptyState | 空状态提示 | 商品列表、搜索结果 |

## 路由跳转关系

```
首页 (/) 
  ├── 点击游戏卡片 ──→ 商品列表 (/goods-list/:gameId)
  ├── 点击搜索框 ────→ 全部游戏 (/games)
  └── 点击热门标签 ──→ 全部游戏 (/games?keyword=xxx)

全部游戏 (/games)
  └── 点击游戏卡片 ──→ 商品列表 (/goods-list/:gameId)

商品列表 (/goods-list/:gameId)
  ├── 点击商品卡片 ──→ 商品详情 (/goods-detail/:goodsId)
  └── 点击搜索框 ────→ 搜索（当前页面筛选）

商品详情 (/goods-detail/:goodsId)
  └── 点击返回 ──────→ 商品列表 (/goods-list/:gameId)
```
